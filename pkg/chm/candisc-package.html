candisc-package(candisc)R Documentation

Generalized Canonical Discriminant Analysis

Description

This package includes functions for computing and visualizing generalized canonical discriminant analyses for a multivariate linear model. They are designed to provide low-rank visualizations of terms in a mlm via the plot method and the heplots package.
